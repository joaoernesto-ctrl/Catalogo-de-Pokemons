# Catalogo-de-Pokemons

Atividade de conclusão de unidade, executada colaborativamente por a equipe de João Pedro de Santana Failla, João Pedro Fagundes Ernesto, Luca Mendes Azevedo e Marcos Saulo Simões da Costa.

---

Catálogo interativo de pokemóns pertecentes a IDs de 1 a 151, sorteados por tipo ou por todos. O site não possui barra de pesquisa, entretanto utiliza-se a busca de personagens por "tipo".
